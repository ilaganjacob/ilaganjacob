## 🚀 Featured Projects

### 🏥 VetMedTeam Platform
**Enterprise continuing education platform for veterinary professionals**

- **Tech Stack:** Angular 19, .NET 8, Azure SQL, Auth0, GraphQL
- **Scale:** 1,000+ active professionals, comprehensive course catalog
- **Architecture:** Clean Architecture with extensive test coverage
- **Key Features:**
  - Single Sign-On (SSO) integration across multiple platforms
  - Real-time course enrollment and progress tracking

**Government cloud infrastructure management system**

- **Tech Stack:** React, .NET, Azure, PostgreSQL
- **Scale:** 500+ government users
- **Key Features:**
  - Azure cost estimation and resource planning
  - PDF export for budget proposals
  - Real-time infrastructure monitoring

**Impact:** Improved cloud resource planning efficiency for government departments
